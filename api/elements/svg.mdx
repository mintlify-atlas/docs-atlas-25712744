---
title: svg()
description: Render SVG graphics in your application
---

`svg()` is used to render SVG (Scalable Vector Graphics) in your app. It accepts SVG data as bytes and renders it using the Skia graphics engine.

## Usage

```rust
use freya::prelude::*;
use bytes::Bytes;

fn app() -> impl IntoElement {
    svg(Bytes::from_static(include_bytes!("icon.svg")))
        .width(48.0)
        .height(48.0)
}
```

## Builder Methods

### SVG Styling

<ParamField path="color" type="Color">
  Set the SVG color (tints the entire SVG).
  
  ```rust
  svg(bytes).color(Color::BLUE)
  ```
</ParamField>

<ParamField path="fill" type="Color">
  Override the fill color of SVG paths.
  
  ```rust
  svg(bytes).fill(Color::RED)
  ```
</ParamField>

<ParamField path="stroke" type="impl Into<Option<Color>>">
  Override the stroke color of SVG paths.
  
  ```rust
  svg(bytes).stroke(Color::BLACK)
  svg(bytes).stroke(None) // Remove stroke
  ```
</ParamField>

### Layout

<ParamField path="width" type="Size">
  Set the SVG width.
  

</ParamField>

<ParamField path="height" type="Size">
  Set the SVG height.
  
  ```rust
  svg(bytes).height(64.0)
  svg(bytes).height(Size::fill())
  ```
</ParamField>

<ParamField path="expanded" type="()">
  Expand to fill available space.
  
  ```rust
  svg(bytes).expanded()
  ```

</ParamField>

<ParamField path="padding" type="Gaps">

  ```rust
  svg(bytes).padding(10.0)
  ```
</ParamField>

<ParamField path="margin" type="Gaps">
  Set margin outside the SVG.
  
  ```rust
  svg(bytes).margin(5.0)
  ```
</ParamField>

<ParamField path="min_width" type="Size">
  Set minimum width.
  
  ```rust
  svg(bytes).min_width(32.0)
  ```
</ParamField>

<ParamField path="min_height" type="Size">
  Set minimum height.
  
  ```rust
  svg(bytes).min_height(32.0)
  ```
</ParamField>

  Set maximum width.
  
  ```rust
  svg(bytes).max_width(128.0)
  ```

</ParamField>

<ParamField path="max_height" type="Size">
  Set maximum height.
  

## Examples

### Basic SVG Icon

```rust
const ICON: &[u8] = include_bytes!("assets/icon.svg");

svg(Bytes::from_static(ICON))
    .width(24.0)
    .height(24.0)
```

### Colored Icon

```rust
svg(Bytes::from_static(ICON))
    .width(32.0)
    .height(32.0)
    .color(Color::from_rgb(100, 150, 255))
```

### Interactive SVG Button

```rust
let mut hovered = use_state(|| false);

svg(Bytes::from_static(ICON))
    .width(48.0)
    .height(48.0)
    .color(if hovered() { Color::BLUE } else { Color::GRAY })
    .on_mouse_enter(move |_| hovered.set(true))
    .on_mouse_leave(move |_| hovered.set(false))
    .on_press(|_| println!("Clicked!"))
```

### Responsive SVG

```rust
svg(Bytes::from_static(LOGO))
    .width(Size::fill())
    .height(Size::px(100.0))
    .padding(20.0)
```

### SVG with Custom Fill and Stroke

```rust
svg(Bytes::from_static(ICON))
    .width(64.0)
    .height(64.0)
    .fill(Color::from_rgb(255, 200, 0))
    .stroke(Color::BLACK)
```

### Rotated SVG

```rust
let angle = use_state(|| 0.0);

use_effect(move |_| {
    // Animate rotation
    angle.set((angle() + 1.0) % 360.0);
});

svg(Bytes::from_static(SPINNER))
    .width(32.0)
    .height(32.0)
    .rotate(angle())
```

### Loading SVG at Runtime

```rust
use bytes::Bytes;

let svg_data = use_state(|| None::<Bytes>);

use_effect(move |_| {
    spawn(async move {
        let bytes = fetch_svg_from_url("https://example.com/icon.svg").await;
        svg_data.set(Some(bytes));
    });
});

if let Some(data) = svg_data() {
    svg(data)
        .width(48.0)
        .height(48.0)
} else {
    label().text("Loading...")
}
```

## Loading SVG Data

### From Static Files

```rust
const SVG_DATA: &[u8] = include_bytes!("icon.svg");
svg(Bytes::from_static(SVG_DATA))
```

### From String

```rust
let svg_string = r#"<svg>...</svg>"#;
svg(Bytes::from(svg_string.as_bytes().to_vec()))
```

### From File at Runtime

```rust
use std::fs;

let svg_bytes = fs::read("path/to/icon.svg").unwrap();
svg(Bytes::from(svg_bytes))
```

## Notes

- SVG rendering is powered by Skia's SVG library
- The SVG is automatically sized based on its viewBox and the width/height you specify
- Color operations tint the entire SVG
- Fill and stroke override the SVG's internal colors
- For best results, use SVGs with proper viewBox attributes
- SVG data must be valid XML
